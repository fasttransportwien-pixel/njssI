# Customer Logos

Replace these placeholder SVG files with the **official brand assets** from your customers
to get pixel-perfect logos.

How to replace:

1. Get the official `.svg` (preferred) or `.png` from the customer.
2. Save it under `/public/logos/` with **the exact same filename** (e.g. `wolt.svg`).
3. Push & redeploy. No code change needed.

If the new file is a PNG, also rename the import in
`src/components/ClientLogos.tsx` from `wolt.svg` to `wolt.png`.
